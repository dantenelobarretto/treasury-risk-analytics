## Overview

Treasury departments play a critical role in managing an organization's liquidity, foreign exchange exposure, and financial risk. However, obtaining realistic treasury datasets for analytics and machine learning is difficult due to confidentiality and regulatory restrictions.

This project develops a synthetic treasury risk dataset that simulates treasury operations, foreign exchange transactions, hedge coverage, cash positions, and financial reporting activities. Using the CRISP-DM (Cross-Industry Standard Process for Data Mining) framework, the project demonstrates how synthetic financial data can support exploratory analysis, predictive modeling, and business intelligence without exposing sensitive information.

## Business Problem

Treasury teams must continuously monitor liquidity, foreign exchange exposure, hedge effectiveness, and financial risk to support strategic decision-making.

Because real treasury datasets are highly confidential, organizations often lack accessible data for analytics development, model experimentation, and employee training.

This project addresses these challenges by generating realistic synthetic treasury data capable of supporting analytics and predictive modeling workflows.

Key challenges addressed include:

- Foreign exchange exposure monitoring
- Treasury cash management
- Hedge effectiveness analysis
- Financial reporting support
- Synthetic financial data generation

---

## Dataset

This project uses a fully synthetic treasury dataset designed to simulate real-world treasury operations.

Dataset includes:

- Treasury transactions
- Foreign exchange positions
- Cash balances
- Hedge coverage information
- Counterparty information
- Currency exposure
- Treasury reporting metrics

The synthetic data preserves realistic financial relationships while ensuring no confidential organizational data is exposed.
